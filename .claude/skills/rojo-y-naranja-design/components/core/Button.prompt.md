Pill-shaped action button — use for any user action; `sand` is the signature hero "RESERVAR" CTA, `primary` (terracotta) for strong actions elsewhere.

```jsx
<Button variant="sand" size="lg">Reservar</Button>
<Button variant="primary">Ver disponibilidad</Button>
<Button variant="outline">Ver detalles</Button>
<Button variant="ghost" size="sm">Cancelar</Button>
```

Variants: `primary` (terracotta fill), `sand` (warm sand pill, dark text — hero CTA), `outline`, `ghost`, `light` (for dark photo overlays). Sizes `sm | md | lg`. Pass `iconLeft` / `iconRight` for icons, `fullWidth` to stretch, `as="a"` for links.
